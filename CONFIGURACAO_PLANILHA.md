2. Adicione o email da conta de serviço: `gestsolicitacao@gestaosolicitacao.iam.gserviceaccount.com`

4. Clique em "Enviar"

### 4. Teste o Sistema
Após configurar as permissões, o sistema deve funcionar normalmente.
